# Metaphor-recognition-and-sentiment
CCAC2026 赛道七：中英文语境下图像隐喻识别和情感分类评测

一、任务背景

本任务聚焦于中英文语境下的多模态社会媒体情感分析，旨在挖掘以“职场女性性别歧视”主题为例的社会话题隐喻识别及情感分类任务。任务依托涵盖中英文语境下的图像数据集，通过解构图像中的隐喻修辞与情感倾向，探讨视觉符号如何传递复杂的社会态度。

本赛道旨在推动计算机视觉与自然语言处理技术在复杂社会话题上的感知能力，促进情感计算与多模态语义对齐技术在社会舆情分析等领域的应用。

本次挑战赛设置两个核心任务：中英文语境下图像隐喻识别与中英文语境下情感分类。

二、任务介绍

（一）任务一：中英文语境下的图像隐喻识别

该任务要求参赛者识别图像中是否存在隐喻表达，并解析其语义映射关系。重点考察模型在职场歧视主题下，对视觉符号（喻体）与其指向的本体之间的逻辑捕捉能力。

输入： 原始图像（中/英文场景）。

输出： 判定是否存在隐喻。

（二）任务二：中英文语境下图像情感分类

该任务要求参赛者根据图像本身的情感表达，结合图像的隐喻含义与表达方式，完成多维度的情感解析。核心在于通过图像所呈现的视觉信息，准确判定其传递的情感极性、强度类别。


输入：原始图像（中/英文场景）及表达方式说明。

输出：情感倾向：消极（1），中性（2）或者积极（3）； 

情感强度：评定情感表达的激烈程度等级，如高级（1）、中级（2）或者低级（3），其中中性的情感强度一般为2。
